The machine folk Session

folk-rnn is a computer model that generates transcriptions in ABC format. This model was trained on about 23,000 transcriptions collected from http://thesession.org. This website provides an interface to this model so that you can create your own new tunes. (More technical information about this model, as well as links to several compositions created using it, can be found at the folk-rnn project webpage.)

The Machine Folk Session is a project funded by the UK Arts and Humanities Research Council, grant no. AH/R004706/1: "Engaging three user communities with applications and outcomes of computational music creativity". See also: folk-rnn. Web applications by Toby Harris.
